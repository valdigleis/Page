Papers

Este manuscrito está sendo construído tendo como base diversas notas de aula que eu preparei para ministrar cursos de graduação nos seguintes tópicos:

  • Matemática Discreta
  • Lógica para Ciência da Computação
  • Linguagens Formais e Autômatos
  • Análise de Algoritmos
  • Computabilidade
  • Teoria dos Números para Computação

Este manuscrito ainda é um projeto em andamento, e possivelmente sua escrita se tornará minha busca particular pelo Graal (nunca sendo realmente concluída), assim é claro que você poderá encontrar diversos erros, que com toda certeza você leitor irá me enviar e-mails ou issues com reports de tais erros.

É importante destacar que este manuscrito não possui qualquer tipo de vínculo editorial, sendo o mesmo ideializado, formatado, escrito e corrigido (com ajuda dos leitores) pelo próprio autor!
